# TextFlip

˙ʎɹɐɹqıʃ ƃuıddıʃɟ ʇxǝʇ ǝʃdɯıs ∀

## Installation

Add `textflip` to your list of dependencies in `mix.exs`:

    ```elixir
    def deps do
      [{:textflip, "~> 0.1.0"}]
    end
    ```
## Usage

```elixir
TextFlip.flip "hello world"
# > "pʃɹoʍ oʃʃǝɥ"
```
